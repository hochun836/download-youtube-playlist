# download-youtube-playlist

## How to Use
- update `resource/playlist/url.json`
- `npm run start`
- output in folder `download`
- log in folder `log`

## How to Test
- update `resource/playlist/url.test.json`
- `npm run test`
- output in folder `download/test`
- log in folder `log/test`
